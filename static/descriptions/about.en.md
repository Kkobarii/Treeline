# About Treeline

## What is Treeline?

Treeline is an interactive learning platform designed to help students understand and visualize data structures and sorting algorithms in real-time. It offers step-by-step animations and interactive simulations that make it easier to understand complex concepts.

## Features

- **Interactive Visualizations:** Watch data structures being built and modified in real-time
- **Step-by-step Animations:** Understand every operation with detailed step-by-step walkthroughs
- **Multiple Data Structures:** Learn binary search trees, AVL trees, red-black trees, and more
- **More Sorting Algorithms:** See how various sorting algorithms work with visualizations and explanations
- **Educational Content:** Detailed descriptions and explanations for each data structure

## Supported Data Structures

Currently, Treeline supports the following data structures:

- Binary Search Tree (BST)
- AVL Tree
- Red-Black Tree
- B-Tree
- Heap
- Linked List
- Stack
- Queue

Recently, I also added sorting algorithms:

- Bubble Sort
- Insertion Sort
- Selection Sort
- Merge Sort
- Quick Sort
- Heap Sort

## Technologies

Treeline is built using modern web technologies including Svelte, TypeScript, Tailwind CSS, and vis.js for visualizations. The whole project is open-source and available on GitHub.
